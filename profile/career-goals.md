# Career Goals

## Short-Term Goal (6-12 Months)

Build strong hands-on expertise in:

- Linux Administration
- Git
- Jenkins
- Docker
- Kubernetes
- Terraform
- AWS

Complete multiple DevOps projects and publish them on GitHub.

---

## Medium-Term Goal (1-2 Years)

Transition into a DevOps Engineer role and contribute to:

- CI/CD pipelines
- Infrastructure Automation
- Cloud Platforms
- Kubernetes Platforms
- Monitoring Solutions

---

## Long-Term Goal (3-5 Years)

Grow into a Senior DevOps / Platform Engineer role specializing in:

- Cloud Infrastructure
- Kubernetes Operations
- Reliability Engineering
- Infrastructure as Code
- Automation at Scale
